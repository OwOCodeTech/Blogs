# 工具推荐

## 系统工具
### Clash
- 跨平台代理工具
- 图形界面配置
- 规则分流支持
- [下载地址](https://github.com/Dreamacro/clash)

### V2rayN
- Windows平台客户端
- 支持多种协议
- 简单易用
- [下载地址](https://github.com/2dust/v2rayN)

## 浏览器插件
### SwichyOmega
- 代理切换工具
- 支持自动切换
- 规则匹配
- [Chrome商店](https://chrome.google.com/webstore)

### uBlock Origin
- 广告拦截工具
- 自定义过滤规则
- 低资源占用
- [Chrome商店](https://chrome.google.com/webstore)
